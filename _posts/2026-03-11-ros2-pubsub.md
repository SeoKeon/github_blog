## 문제

센서 노드와 제어 노드 간 메시지를 느슨하게 연결하고 싶다.

## 개념 설명

ROS2는 topic 기반 pub/sub 모델을 사용한다. publisher는 데이터를 발행하고 subscriber는 동일 topic을 구독한다.

## Python 코드

```python
# publisher 핵심 아이디어
# node.create_publisher(String, 'robot_status', 10)
# publisher.publish(msg)

# subscriber 핵심 아이디어
# node.create_subscription(String, 'robot_status', callback, 10)
```

## 결과

노드 간 결합도를 낮추고, 분산 시스템 구성이 쉬워진다.

## 정리

ROS2 pub/sub는 로봇 소프트웨어 아키텍처의 기본 통신 패턴이다.
