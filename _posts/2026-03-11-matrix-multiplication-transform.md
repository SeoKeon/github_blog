## 문제

기준 좌표계가 다른 위치/자세 정보를 한 좌표계로 통합하고 싶다.

## 개념 설명

좌표변환은 회전과 이동의 조합이며, 동차변환행렬 곱으로 연쇄 프레임을 연결한다.

## 수식

$$
{}^{0}T_{2} = {}^{0}T_{1}\ {}^{1}T_{2}
$$

## Python 코드

```python
import numpy as np

T01 = np.array([[1, 0, 1],
                [0, 1, 2],
                [0, 0, 1]])
T12 = np.array([[0, -1, 0],
                [1,  0, 1],
                [0,  0, 1]])

T02 = T01 @ T12
print(T02)
```

## 결과

프레임이 늘어나도 동일한 방식으로 pose를 누적 계산할 수 있다.

## 정리

행렬곱은 단순 산술이 아니라 "프레임 이동 기록"이다.
